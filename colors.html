HTML Colors

HTML colors are specified with predefined color names, or with RGB, HEX, HSL, RGBA, or HSLA values.


Color Names

In HTML, a color can be specifiedd by using a color name:

  • Tomato
  • DodgerBlue
  • Gray
  • Violet
  • Orange
  • MediumSeaGreen
  • SlateBlue
  • LighGray
HTML supports 140 standard color names.
